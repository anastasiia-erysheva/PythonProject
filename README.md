# Решения задач из курса «Поколение Python»

Репозиторий содержит мои практические решения задач по программированию на Stepik. Код структурирован и отражает прогресс в изучении продвинутых концепций языка.

## 🛠 Технологии
* **Язык:** Python 3.12
* **Среда разработки:** PyCharm

## 📚 Изученные темы и модули

### 1. Продвинутые структуры данных (`collections`)
* **`Counter`** — автоматический подсчет элементов, поиск самых частотных значений (`most_common`).
* **`defaultdict`** — словари со значениями по умолчанию и обработка пропущенных ключей (`setdefault`).
* **`namedtuple`** — создание именованных кортежей для структурирования объектов.
* **`OrderedDict`** — упорядоченные словари и сортировка элементов.

### 2. Сериализация и хранение данных
* **`json`** — работа с текстовым форматом данных, чтение, запись и парсинг конфигурационных файлов (`profile.json`).
* **`pickle`** — бинарная сериализация объектов Python, сохранение и восстановление состояний.

### 3. Работа с файловыми архивами (`zipfile`)
* Чтение архивов, извлечение метаданных (`infolist`), работа со временем изменения файлов и программная запись/упаковка данных в `.zip`.

### 4. Ввод-вывод и базовые типы
* Потоковый ввод данных через системный модуль **`sys`** (`stdin`).
* Работа со встроенными коллекциями: списки (`list`), множества (`set`) и словари (`dict`).
### 5. Работа с датой и временем
* **`datetime`** — работа с датами, временем и временными интервалами.
* **`calendar`** — работа с календарем, определение высокосных лет и вывод календарей на экран.
